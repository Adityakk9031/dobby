# Dobby Drive Agents

This repository is a single Next.js app designed for Vercel deployment.

- `app/`: routes, pages, layouts, and route handlers
- `components/`: reusable UI pieces
- `contexts/`: client-side providers
- `lib/`: auth, database, utilities, and server helpers
- `models/`: shared TypeScript models for the drive domain
- `prisma/`: Neon/Postgres schema
